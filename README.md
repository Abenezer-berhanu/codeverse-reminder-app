## Local Setup

1. Clone the repository
2. Copy `.env.example` to `.env`
3. Update database credentials in `.env`
4. Run migrations

If using Laravel Herd:

- Visit http://project-name.test

If not using Herd:

- Run `npm run dev`
- Visit http://localhost:8000 or the url with name APP_URL: http://localhost
